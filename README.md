## 🧠 Internship Overview

This repository serves as the **master portfolio** for my **Data Analyst Internship at ApexPlanet**.  
It consolidates all tasks completed during the internship and demonstrates an **end-to-end data analytics workflow**, from raw data handling to business storytelling and statistical validation.

The portfolio reflects hands-on experience with **real-world data**, focusing on data quality, analysis, visualization, SQL querying, and insight communication.

During this internship, I completed 5 structured tasks focusing on:

Each task represents a key data analytics skill in the lifecycle of a real business problem.
