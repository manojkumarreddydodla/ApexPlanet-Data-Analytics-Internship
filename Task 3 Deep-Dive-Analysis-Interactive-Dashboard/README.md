# 📊 Deep-Dive Analysis & Interactive Dashboard

## 📌 Project Overview
This project solves complex business problems using KPI framework, cohort analysis, and an interactive BI dashboard.

Business Context: E-commerce platform (2 years data)

---
 
## 🎯 Core KPIs

1. Conversion Rate = (Orders / Sessions) * 100  
2. Customer Churn Rate  
3. Average Order Value (AOV)  
4. Customer Lifetime Value (CLV)  
5. Retention Rate  

---

## 🧹 Data Preparation

- Cleaned missing values
- Removed duplicates
- Handled outliers
- Derived cohort month
- Calculated RFM metrics

---

## 🔍 Deep-Dive Analysis (Cohort Analysis)

Key Findings:
- Q1 cohorts retain 8–10% better
- Email channel generates highest CLV
- Paid ads customers churn faster
- Retention drops significantly after Month 2

Business Recommendation:
- Improve loyalty programs
- Invest in email marketing
- Optimize paid ad targeting

---

## 📊 Dashboard Features

- KPI Cards with trends
- Revenue by region & channel
- Cohort retention heatmap
- Filters (Date, Region, Channel)
- Drill-through analysis
